# Воркшоп по разработке через тестирование на TechLeadConf

## Требования
Пишем банковский счёт
1. Я могу положить деньги на счёт
1. Я могу снять деньги со счёта
1. Я могу получить выписку со счёта (дата + баланс)
1. Я могу переводить деньги со счёта на счёт

## Python

Для работы нужен `pipenv`

1. Установи зависимости: `pipenv install --dev`
1. Тест пишем в `src/account_test.py`
1. Код пишем в `src/account.py`

В зависимостях есть `pytest`, проще всего использовать его
